## 摘要
基于 Transformer 的 SMILES 语言模型存在局部性差距：标准字符级分词会割裂化学上有意义的基序，迫使模型以牺牲长程依赖为代价反复学习局部语法。为解决此问题而不破坏标准分词器，我们提出了 MolGram，它将条件 n-gram 记忆模块集成到分子语言模型中。MolGram 通过可扩展的哈希查找将局部字符串模式映射为学习嵌入，并将此区域上下文动态注入隐藏状态。在包括无条件分子生成、正向反应预测和单步逆合成在内的三个任务上的评估显示，MolGram 持续提升了性能。关键的是，我们的分析表明，MolGram 优于参数数量多 3 倍的基线模型，确立了显式局部模式记忆作为一种高效归纳偏置的地位。

## Abstract
Transformer-based language models for SMILES strings suffer from a locality gap: standard character-level tokenization fragments chemically meaningful motifs, forcing models to repeatedly learn local syntax at the expense of long-range dependencies. To address this without disrupting standard tokenizers, we propose MolGram, which integrates a conditional $n$-gram memory module into molecular language models. MolGram maps local string patterns to learned embeddings via scalable hash lookups and dynamically injects this regional context into hidden states. Evaluations across three tasks, including unconditional molecule generation, forward reaction prediction, and single-step retrosynthesis, show that MolGram consistently improves performance. Crucially, our analyses demonstrate that MolGram outperforms baselines with 3$\times$ more parameters, establishing explicit local pattern memory as a highly efficient inductive bias.

## 2. 方法论

### 核心思想
- 在标准 Transformer 分子语言模型的基础上，插入一个 **条件 n-gram 记忆模块（conditional n-gram memory module）**，让模型能够直接存储并利用固定长度窗口内的局部字符串模式（local string patterns），而不是仅依靠自注意力机制从原始 token 中反复学习这些模式。
- 该模块 **不改变已有的分词器**，可以方便地集成到各种基于 SMILES 的预训练或任务模型中。

### 关键技术细节
- **n-gram 模式提取**：对于输入 SMILES 中的每个位置，以当前 token 为中心（或附近窗口）提取长度为 n 的字符序列（local n-grams）。
- **可扩展哈希映射**：利用哈希函数（scalable hash lookups）将这些局部字符串模式映射到一个固定规模的嵌入表（embedding table）中。该设计避免了显式构建庞大的 n-gram 词表，保持了内存和计算的可控性。
- **动态注入隐藏状态**：从嵌入表中查得的 n-gram 嵌入被动态地注入到 Transformer 层（例如加在自注意力或前馈网络的输入/输出上），为每个位置的隐藏状态提供关于其周围局部上下文（regional context）的显式信息。
- **整体流程**（文字描述）：
  1. 输入 SMILES 经字符级分词器与嵌入层，得到初始 token 表示。
  2. 并行计算每个位置的 n-gram 哈希值，从可学习的记忆矩阵中提取对应的局部上下文嵌入。
  3. 在若干 Transformer 层（可为所有层或特定层）中，将 n-gram 嵌入与原始隐藏表示融合（如相加或拼接）。
  4. 后续层继续基于增强后的表示进行推理，最终输出生成或预测结果。

该方法可视为一种 **轻量级、模块化的归纳偏置**，无需改动模型主干即可强化局部语法感知。
